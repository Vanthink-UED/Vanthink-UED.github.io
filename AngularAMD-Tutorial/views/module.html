创建 Module

目前 angularAMD 支持的模块内部定义的方法有:

  • .provider

  • .controller

  • .factory

  • .service

  • .constant

  • .value

  • .directive

  • .filter

  • .animation

你可以快速写一个 fatory

define(['app'], function (app) {
    app.factory('Pictures', function (...) {
        ...
    });
});
